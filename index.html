PROJECT

Sketch to Wireframe

2022.04 - 2022.05

UI 스케치를 빠르고 쉽게 Wireframe으로 변환하는 도구를 개발하는 것을 목표로 한 프로젝트입니다. 이를 위해 프론트엔드, 백엔드, 그리고 인공지능 분야에서 각각의 역할을 수행하였습니다.

프론트엔드
  • TypeScript와 React를 활용하여 프론트엔드를 개발
  • 객체 검출 결과와 Material UI 컴포넌트를 매핑하여 Wireframe 생성 기능을 구현
백엔드
  • Python과 Django를 활용하여 백엔드를 개발
  • UI 스케치를 모델에 전달하고, 객체 검출 결과를 프론트엔드에 전달하는 기능을 구현
인공지능
  • 기존 모델을 사용하여 전이학습을 통해 객체 검출 모델을 생성
  • 사용 모델 - Faster RCNN / RetinaNet / TOOD / YOLOF / YOLOX

이 프로젝트에서 가장 큰 어려움은 인공지능 분야에서 전이학습을 통해 좋은 성능을 내는 모델을 만들지 못한 것이었습니다. 하지만 각 분야의 기능을 구현하고 전체적인 프로그램을 구성함으로써, 각 분야가 상호작용하는 방식과 전체적인 프로그램 구조를 배울 수 있는 좋은 기회가 되었습니다.
